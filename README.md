## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
# mySQL server should be turned on.
$ yarn start:local

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
