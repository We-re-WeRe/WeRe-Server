# WeRe-Server

### 🚀 Latest version: 0.0.0

## Description

WeRe의 Server side Repository입니다.  
웹툰을 좋아하는 사람들을 위한 추천, 리뷰 확인 플랫폼을 제공합니다.

## Tech.

- NodeJS v20.11.1
- npm v6.14.17
- NestJS v9.1.5
- mysql v8.3.0
- jest v28.1.3

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
# mySQL server should be turned on.
$ yarn start:local

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
